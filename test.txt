
[考察知识点包含但不仅限于：FileReader读取文件内容，JSON解析，数据类型，集合，排序，数组，异常处理]

完成以下题目，并编写单元测试进行自测。
1、读取Account.json文件内容，转换为用户账号对象。
读取Account.json文件内容后，完成如下4题。

1、根据账号创建日期降序排列所有账号信息。

2、统计所有用户对应的手机号码。

3、统计所有账号状态，用户类型默认为新用户。
（1）统计状态为在线的账号上次登录日期，计算这些用户当前日期与上次登录日期的间隔天数。

将间隔天数为2天及以下的用户类型设置为活跃用户。
将间隔天数为100天及以上的用户设置为回访用户。

（2）统计状态为离线的账号上次登录日期，计算这些用户当前日期与上次登录日期的间隔天数。

将间隔天数为100天及以上的用户设置为流失用户。

4、统计所有用户年龄分布：
（1）以每10岁为一个区间（0-9，10-19，...），统计每个区间的年龄人数。
（2）统计所有用户的年龄，去重升序排序。
（3）统计同龄用户个数。


Account对象说明：
Account: {
  username: 用户名 String,
  password: 密码 String,
  age: 年龄 Integer,
  email: 邮箱 String,
  phone: 手机 String,
  type: 类型 enum 新用户、活跃用户、流失用户、回访用户,
  status: 状态 enum,
  createDate: 创建日期 Date,
  1上次登录日期 Date
}
